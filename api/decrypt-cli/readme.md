# Build
`cargo build --release`

# How to use
- requirement: encrypted-graph-data, contact the Ideamesh team to fetch it.

```
> <path-to-rsapi-repo>/target/release/decrypt-cli -h
Usage: decrypt-cli [OPTIONS] --pwd <password> --dir <PATH>

Options:
      --pwd <password>  graph password
      --dir <PATH>      graph data dir path
      --dst <PATH>      dir to store decrypted data [default: ./decrypted]
  -h, --help            Print help
```
